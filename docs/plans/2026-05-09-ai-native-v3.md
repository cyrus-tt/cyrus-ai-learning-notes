# AI Native 个人站 v3 — 数字分身 + 生成式艺术 + MCP

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 把 cyrustyj.xyz 升级为 AI Native 个人站：AI Cyrus 数字分身聊天、p5.js 生成式艺术 hero、View Transitions 丝滑导航、llms.txt + MCP 端点让 AI Agent 可发现。

**Architecture:** Cloudflare Workers AI 做 LLM 推理，知识库用 JSON 预提取（v1 简单方案），前端浮动聊天面板。生成式艺术用 p5.js 纯 Canvas。MCP 用 Workers Functions 暴露结构化 API。

**Tech Stack:** Cloudflare Workers AI, p5.js, vanilla JS, Python (build scripts)
