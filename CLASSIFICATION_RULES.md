# AI 产业上中下游分类定义

用于 `scripts/update_news.py` 的 `industryStage` 字段。

## 上游（基础设施层）

- 定义：为 AI 提供算力和底层基础设施的环节。
- 典型对象：GPU、芯片、服务器、数据中心、云基础设施、网络互联。
- 关键词示例：`chip`、`gpu`、`nvidia`、`amd`、`算力`、`芯片`、`数据中心`。

## 中游（模型与平台层）

- 定义：模型研发、训练、推理平台、开发框架与 API 能力。
- 典型对象：LLM、基础模型、多模态模型、SDK、API、评测框架、MCP 等。
- 关键词示例：`model`、`llm`、`mcp`、`api`、`training`、`benchmark`、`大模型`。

## 下游（应用与场景层）

- 定义：面向具体业务场景和终端用户的 AI 应用落地。
- 典型对象：内容生产、营销、客服、教育、医疗、办公协同、业务自动化。
- 关键词示例：`application`、`workflow`、`productivity`、`应用`、`落地`、`自动化`。

## 当前判定方式

- 脚本采用关键词命中计分，按命中最高分类。
- 若分值并列，优先级：`上游` -> `中游` -> `下游`。
- 若无有效命中，默认归到 `中游`。
