6 tools in rotation

我的 AI 工具栈

不追新、不囤工具。每一个都解决一个具体问题，合在一起就是从想法到产品的完整链路。

主力编程

Claude Code

写功能、改 bug、重构、部署 — 这个网站就是用它从零搭的。终端里跑，不用切 IDE，直接在项目上下文里工作。

$ claude "给网站加一个工具栈页面"
+ tools.html created
+ 25 files nav updated
思考伙伴

Claude

梳理产品逻辑、写方案、做决策分析。不是搜索引擎，是能跟你辩论的同事。

代码审查

Codex

代码写完丢给它做独立审查，能抓到自己看不到的盲点。异步跑，不占主线程。

知识库

Obsidian

读书笔记、项目复盘、灵感碎片全在这。配合 Claude Code 变成可编程的第二大脑。

个人分身

Hermes

学了我的风格和知识体系，能代替我回答常见问题、处理日常沟通。7 x 24 在线。是所有工具中最"懂我"的一个。

自动化

AutoCLI

把重复的终端操作编排成工作流 — 部署、数据同步、定时任务，一条命令搞定。

typical workflow

Claude 想 → Claude Code 写 → Codex 审 → AutoCLI 部署 → Hermes 交付
