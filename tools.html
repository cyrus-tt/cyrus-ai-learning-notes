我的 AI 工具栈

daily driver stack

不追新、不囤工具。这 6 个是真正每天在用的，每一个都解决一个具体问题。

⚡

Claude Code

主力编程搭档。写功能、改 bug、重构、部署 — 这个网站就是用它从零搭的。终端里跑，不用切 IDE。

编程
🧠

Claude

思考伙伴。梳理产品逻辑、写方案、做决策分析。不是搜索引擎，是能跟你辩论的同事。

思考
🔍

Codex

Review 帮手。代码写完丢给它做独立审查，能抓到自己看不到的盲点。异步跑，不占主线程。

审查
📚

Obsidian

本地知识库。读书笔记、项目复盘、灵感碎片全在这。配合 Claude Code 变成可编程的第二大脑。

知识管理
⚙️

AutoCLI

自动化中枢。把重复的终端操作编排成工作流 — 部署、数据同步、定时任务，一条命令搞定。

自动化
🤖

Hermes

个人 AI 分身。学了我的风格和知识体系，能代替我回答常见问题、处理日常沟通。7×24 在线。

分身
