// 防抖 const debouncedSearch = Utils.debounce((query) => { console.log('搜索:', query); }, 300); // 节流 const throttledScroll = Utils.throttle(() => { console.log('滚动事件'); }, 300); // 日期格式化 Utils.formatDate(new Date(), 'YYYY-MM-DD HH:mm:ss'); Utils.timeAgo('2026-03-14T10:00:00'); // 复制到剪贴板 await Utils.copyToClipboard('要复制的文本');

⚡ 性能优化建议

  • ✅ 使用 Toast 替代 alert()，避免阻塞用户操作
  • ✅ API 请求时显示 Loading 状态，提升用户体验
  • ✅ 搜索、滚动等高频操作使用防抖/节流
  • ✅ 大列表使用骨架屏，减少白屏时间
  • ✅ 图片添加 loading="lazy" 属性
  • ✅ 避免在循环中进行 DOM 操作
